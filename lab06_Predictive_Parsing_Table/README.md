# Lab 06 – Predictive Parsing Table

## Aim
To construct a predictive parsing table for a given grammar.

## Algorithm
1. Read grammar productions.
2. Identify terminals and non-terminals.
3. Initialize parsing table.
4. Fill table using grammar productions.
5. Display predictive parsing table.

## How to Compile
gcc lab06_Predictive_Parsing.c -o lab6

## How to Run
./lab6

## Output
Displays the predictive parsing table.
