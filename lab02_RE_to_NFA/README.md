# Lab 02 – Conversion from Regular Expression to NFA

## Aim
To convert a given Regular Expression into its equivalent NFA.

## Algorithm
1. Read the regular expression.
2. Process each symbol of the expression.
3. Construct NFA using Thompson’s construction.
4. Display NFA transitions.

## How to Compile
gcc re_to_nfa.c -o lab2

## How to Run
./lab2
