# Lab 08 – Computation of LEADING and TRAILING

## Aim
To compute LEADING and TRAILING sets for a given grammar.

## Algorithm
1. Read grammar productions.
2. Identify non-terminals.
3. Compute LEADING set for each non-terminal.
4. Compute TRAILING set for each non-terminal.
5. Display the LEADING and TRAILING sets.

## How to Compile
gcc lab08_Leading_Trailing.c -o lab8

## How to Run
./lab8

## Output
Displays LEADING and TRAILING sets for the grammar.
