# Lab 11 – Intermediate Code Generation (Quadruple, Triple, Indirect Triple)

## Aim
To generate intermediate code in the form of Quadruple, Triple, and Indirect Triple representations.

## Algorithm
1. Read the number of expressions.
2. For each expression, read operator, operands, and result.
3. Generate the Quadruple table.
4. Generate the Triple table.
5. Generate the Indirect Triple table.
6. Display all the tables.

## How to Compile
gcc lab11_Intermediate_Code_Generation.c -o lab11

## How to Run
./lab11

## Output
Displays Quadruple, Triple, and Indirect Triple representations of the given expressions.
