# Lab 12 – Simple Code Generator

## Aim
To generate simple target code for arithmetic expressions.

## Algorithm
1. Read the number of statements.
2. For each statement, read operator, operands, and result.
3. Generate simple assembly-like instructions using registers.
4. Display the generated target code.

## How to Compile
gcc lab12_Simple_Code_Generator.c -o lab12

## How to Run
./lab12

## Output
Displays simple target code for the given arithmetic expressions.
