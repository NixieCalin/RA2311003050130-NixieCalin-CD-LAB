# Lab 05 – FIRST and FOLLOW Computation

## Aim
To compute the FIRST and FOLLOW sets for a given grammar.

## Algorithm
1. Read grammar productions.
2. Identify terminals and non-terminals.
3. Compute FIRST sets for all non-terminals.
4. Compute FOLLOW sets using FIRST sets.
5. Display FIRST and FOLLOW sets.

## How to Compile
gcc lab05_FIRST_FOLLOW.c -o lab5

## How to Run
./lab5

## Output
Displays FIRST and FOLLOW sets of the grammar.
