# Lab 03 – Conversion from NFA to DFA

## Aim
To convert a given NFA into its equivalent DFA.

## Algorithm
1. Take the NFA as input.
2. Apply subset construction method.
3. Create DFA states from NFA state combinations.
4. Generate DFA transition table.

## How to Compile
gcc nfa_to_dfa.c -o lab3

## How to Run
./lab3
