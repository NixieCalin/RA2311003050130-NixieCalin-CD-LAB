# Lab 04 – Elimination of Ambiguity, Left Recursion and Left Factoring

## Aim
To eliminate ambiguity, remove left recursion and perform left factoring for a given grammar.

## Algorithm
1. Read grammar productions from the user.
2. Display the given grammar.
3. Identify left-recursive productions.
4. Eliminate left recursion.
5. Display the transformed grammar.

## How to Compile
gcc lab04_Grammar_Transformation.c -o lab4

## How to Run
./lab4

## Output
The program displays the grammar after eliminating left recursion.
