# Lab 15 – Storage Allocation Strategy (Stack)

## Aim
To implement stack storage allocation strategy.

## Algorithm
1. Initialize stack and top pointer.
2. Perform push operation to allocate memory.
3. Perform pop operation to deallocate memory.
4. Display stack contents.
5. Repeat operations until exit.

## How to Compile
gcc lab15_Storage_Allocation.c -o lab15

## How to Run
./lab15

## Output
Demonstrates stack allocation using push and pop operations.
