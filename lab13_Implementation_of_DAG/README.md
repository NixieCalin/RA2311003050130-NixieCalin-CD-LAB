# Lab 13 – Implementation of DAG

## Aim
To implement Directed Acyclic Graph (DAG) representation for arithmetic expressions.

## Algorithm
1. Read the number of expressions.
2. For each expression, read operator, operands, and result.
3. Construct the DAG representation.
4. Display the DAG table.

## How to Compile
gcc lab13_Implementation_of_DAG.c -o lab13

## How to Run
./lab13

## Output
Displays the DAG representation of the given expressions.
