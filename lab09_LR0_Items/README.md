# Lab 09 – Computation of LR(0) Items

## Aim
To compute LR(0) items for a given grammar.

## Algorithm
1. Read grammar productions.
2. For each production, place the dot at all possible positions.
3. Generate LR(0) items.
4. Display the LR(0) items.

## How to Compile
gcc lab09_LR0_Items.c -o lab9

## How to Run
./lab9

## Output
Displays all LR(0) items for the given grammar.
